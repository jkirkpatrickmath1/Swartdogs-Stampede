# Swartdog Stampede

Welcome to **Swartdog Stampede**, the Human FRC Game Simulation for FRC Team 525 – the Swartdogs.  
This project allows you to simulate matches, track alliances, score matches, and display running tickers, all using a fully branded interactive website.

---

## **Project Overview**

- Web-based simulation of the Swartdog Stampede FRC game.
- Built with **Flask**, HTML, CSS, and JavaScript.
- Supports:
  - Match scheduling
  - Real-time match countdown with start/pause/replay
  - Audio cues (charge, warning, buzzer)
  - Score submission for Red and Blue alliances
  - Bottom ticker showing upcoming matches
  - Embedded logos and team branding

---

## **Branding**

- **Colors**: Black (#000000) and Swartdog Orange (#FF6E00)  
- **Fonts**:  
  - Bolt (Headings – ALL CAPS)  
  - Roboto (Body / Text)  
  - Roboto Mono (Technical UI / Ticker / Inputs)  
- **Logos**:  
  - `logo.png` – standard logo  
  - `longLogo.png` – banner logo  

---

## **Features**

- **Match Page**:  
  - Start, Pause, Replay buttons  
  - Real-time countdown timer  
  - Sound effects for match events  
  - Submit scores for Red and Blue alliances  
- **Downloads Page**:  
  - PDF files: Rules, Facilitator Guide, Score Sheet  
- **Video Page**:  
  - MP4 or embedded YouTube video of the game  
- **Ticker**: Shows upcoming matches with team alliances
